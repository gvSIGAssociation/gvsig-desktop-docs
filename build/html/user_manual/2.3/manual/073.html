73. Plantillas de Mapa¶

71.1. Plantillas de Mapa

Una plantilla de Mapa es un archivo .gvslt que contiene un diseño de un Mapa que se puede utilizar como referencia para crear nuevos Mapas. Las plantillas de Mapa facilitan la reutilización o la normalización de un diseño en una serie de Mapas. El uso de una plantilla permite ahorrar tiempo, puesto que no es necesario diseñar manualmente una y otra vez las partes comunes de los Mapas.

Estas herramientas están disponibles desde el menú “Mapa“.
